## üé® Screenshot Tips

**For Clean Screenshots:**
1. Use a clean terminal (PowerShell or Windows Terminal)
2. Maximize window for readability
3. Use default color scheme (or dark theme if preferred)
4. Clear scrollback before each command: `cls`
5. Take full window screenshots (not just cropped portions)

**Recommended Tools:**
- Windows Snipping Tool (Win + Shift + S)
- ShareX (free, advanced)
- Greenshot (free, with annotations)

**File Naming:**
- `screenshot_1_username_search.png`
- `screenshot_2_html_report.png`
- `screenshot_3_resources_list.png`
- `screenshot_4_auto_search.png`
- `screenshot_5_full_investigation.png`
